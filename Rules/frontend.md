# Rules — Frontend

## General Workflow Rules

### Rule 1: Release Notes Entry Required
When code is **committed**, an entry **must** be added to `Release notes/release-notes.md` with the following details:
- **Prompt Given** — the exact prompt provided
- **Activities** — a concise list of all changes made
- **Commit ID** — the git commit hash and message

This applies to all AI models (GitHub Copilot, Claude Code, Cursor, Windsurf, etc.).
